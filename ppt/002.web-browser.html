  • 浏览器内核

    哪些浏览器的内核是一样的,或者类似的
    1. Trident内核：IE最先开发或使用的，也称IE内核
    2. Webkit内核：前身是KHTML, Apple发扬光大, Chrome(blink) 让它风靡, Opera 已跟风
    3. Gecko内核： Netscape6开始采用的内核，后来的Mozilla FireFox (火狐浏览器) 也采用了该内核
    4. Presto内核：以前只有Opera浏览器采用该内核

  • 浏览器组成结构熟悉你的浏览器

    浏览器一般都具有哪些重要组成
    1. 网页可视窗体( 视区, 滚动条 )
    2. 地址栏, 导航栏, 标签, 书签
    3. 前进, 后退, 刷新/停止刷新
    4. 插件区域( webkit )
    5. 设置 - 系列功能

  • Webkit调试熟悉Webkit的控制台

    1. element 页面元素选择
    2. network 网络相关功能
    3. sources 所有当前页面、子页面、插件资源
    4. timeline 浏览器系统监控(☆)
    5. profiles 性能监控
    6. resources 本地存储
    7. audits 网页可优化方案检测
    8. console 调试控制台(☆☆☆☆☆)
