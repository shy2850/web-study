  • head 标签

    网页全局信息承载体
    1. title 定义文档标题。
    2. base 定义页面上所有链接的默认地址或默认目标。
    3. link 定义文档与外部资源之间的关系。
    4. meta 定义关于 HTML 文档的元数据。
    5. script 定义客户端脚本。
    6. style 定义文档的样式信息。

  • HTML书写注意事项

    应该遵循规范 (☆☆☆☆☆)
    1. 标准的HTML整体结构
    2. meta定义编码并前置
    3. 如无特殊说明, 所有的html标签名、属性名、简单属性值, 均使用小写
    4. 标签内部文本的 < > 等字符均使用&lt; &gt;或者相应的实体符号代替
    5. 组合标签内, 杜绝滥竽充数
    6. 标签嵌套 [div, form, li] > [h1-h6, p] > [a, span] > [img, input]
