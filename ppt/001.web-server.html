• 简介什么是服务器？

    在前端看来就是能够通过http(s)协议访问资源的应用程序
    1. 可以使用http路径访问html页面， css， js， 图片等资源
    2. 方便前端进行一些交互性的功能调试，如：ajax
    3. 更接近真实运行环境的调试效果
    4. 了解一些服务端代码编写，方便数据调试和前端构建 (☆)
