# 简单Dialog组件

## 基本功能
- 基本组件
	- 半透明遮罩层 Mask
	- 弹出框容器 Holder
		- 内容头部 head
		- 内容主体 body
		- 底部操作 foot
- 组件功能
	- 模拟 alert, confirm, prompt 原生效果
	- 在回调中获取操作结果
	- 支持Priomise写法

## 代码描述
    - css/dialog.less & css/dialog-ie6.less
    - base-01.html 基本HTML结构
    - base-02.html 基本js构建HTML 和 绑定事件
    - base-03.html 封装js构建HTML 和 绑定事件
